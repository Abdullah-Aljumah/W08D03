## Dependencies

- Dotenv
  to hide our content keys

```bash
npm i dotenv
```

- Express
  to build our server

```bash
npm i express
```

- bcrypt
  to bcrypt our passwords

```bash
npm i bcrypt
```

- jsonwebtoken
  genreate tokens

```bash
npm i jsonwebtoken
```

- Mongoose
  to save our data
  here the website [Mongoose](https://account.mongodb.com)
